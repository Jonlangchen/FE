# VSCode 中 Git 的配置
## 可能有些人打开这里发现明明安装了git为什么提示没有安装或者检测不到，因为你安装的时候修改了默认的安装路径所以这里就检测不出来了。
    解决：文件—首选项—配置，然后在搜索框中输入git.path，然后打开相应的配置文件，把你的安装路径填入”git.path”:”F:\\Git\\bin\\git.exe”中。

    注意：git 必须配置系统变量 path。
